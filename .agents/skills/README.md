# Codex Skills 白名单

当前仓库级 Codex skill 只保留：

- `repo-module-admission`：新增模块、skill、hook 或 tool 前的准入判断。
- `repo-verification-before-completion`：任务完成前验证。

不默认新增其他 skill。新增 skill 必须先得到用户明确要求。

不保留：

- review 类 skill
- maintenance 类 skill
- brainstorm 类 skill
- learning / memory promotion
- worktree / PR / resume 高权限 skill
