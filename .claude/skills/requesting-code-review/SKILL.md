# requesting-code-review

Use this skill when packaging work for review by another human or agent.

## Include

- What changed in behavior or contract.
- What risks or regressions remain.
- What verification was run and what passed.
- Any open questions that would change review conclusions.

## Exclude

- Empty praise.
- Invented workflow steps.
- Mandatory branch or worktree policy.
