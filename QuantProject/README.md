# QuantProject (全资产量化仓位决策系统)

## 📌 项目简介

这是一个自动化的多资产量化投资决策系统（2026 优化版）。系统旨在通过固定的策略模型和技术指标（如移动均线、波动率偏离度等），对包含美股、日股、黄金和加密货币在内的多种核心资产进行资金分配与仓位建议。系统包含自动数据拉取、增量处理和本地离线决策报告生成。

## 🎯 核心资产及资金分配

系统目前的标准资金配置比例如下：
* **SPY (标普 500 ETF)**：21.25%
* **QQQ (纳斯达克 100 ETF)**：21.25%
* **EWJ (日本股市 ETF)**：21.25%
* **XAU (现货黄金)**：21.25%
* **BTC (比特币)**：15.00%

## 📂 核心文件架构

* **`start.bat`**：系统的主入口脚本。负责环境检查、自动创建目录以及一键依序运行「数据更新」和「引擎决策」环节。
* **`config.py`**：系统全局配置文件。配置项包含相对路径自动解析 (`BASE_DIR`)、金融平台代码映射（`Stooq` 和 `yfinance` 代码字典）以及上述的仓位权重配置。
* **`update_stooq_fast.py`**：高速多线程数据更新模块。默认优先通过高性能的 `yfinance` 接口抓取全量市场数据，若受到网络限制则自动平滑降级到 `Stooq` 备用接口。自动执行增量更新和本地数据存储。
* **`decision_engine.py`**：行情回溯与仓位决策引擎。程序根据各个资产设定的独立算法模型计算出最终的投资百分比，并支持用户输入“实盘总资金”来自动计算具体的实盘调仓金额。

## 🧠 量化策略模型一览

1. **SPY (Sigmoid Smooth 策略)**
   使用 12 个月移动平均线 (MA12) 的偏离度作为输入，通过 Sigmoid 函数平滑计算权重。偏离度在 ±10% 时触发强多/空头规则。
2. **QQQ & EWJ (Combo Adaptive 策略)**
   计算多周期移动平均（MA3/6/9/12）结合短期波动率（Vol）计算出动量因子与波动率调整 Alpha，内建动量崩盘过滤机制防范暴跌风险。
3. **XAU (Trend Discrete 策略)**
   使用 MA6、MA9、MA12、MA15 进行多级别离散趋势判定，跌破均线会阶梯式减仓或清仓。
4. **BTC (Trend Fast MA6 策略)**
   采用快速均线反应（MA6），根据价格是否超过该均线判定是全仓干顸还是空仓防守。

## 🚀 运行方式

1. 确保安装了最新版 Python 并在终端运行 `pip install pandas numpy requests yfinance` 补全环境依赖。
2. 双击运行（或在终端运行）目录下的 `start.bat` 文件。
3. 系统将自动执行：
   * 检查和创建 `data` 目录。
   * 通过多线程同步更新五大资产的本地数据。
   * 运行决策引擎，如果终端提示，可以通过输入您的「实盘总资金」来进行精确的建仓金额规划（回车默认纯百分比显示）。
   * 控制台输出决策报告图表，并将结果持久化追加归档至 `position_history.txt` 记录文件。

## ⚙️ 常见问题 (FAQ)
* **下载卡住或特别慢怎么办？**：此版已通过 `yfinance` 优化，通常能秒速完成数据下载。如遇网络异常，会自动转为 Stooq 备用，稍加等待即可。
* **可以增减交易资产吗？**：可以通过在 `config.py` 里的 `ASSETS_MAPPING` 和 `ALLOCATION_WEIGHTS` 字段进行调整。需同步在 `decision_engine.py` 里添加处理逻辑。
