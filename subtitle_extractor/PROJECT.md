# 视频字幕批量提取工具

## 项目概述

本项目旨在提供一个简单易用的工具，用于从无字幕视频中自动提取音频并使用 faster-whisper 模型生成带时间戳的 Markdown 格式字幕。该工具支持批量处理视频文件，适用于内容创作者、研究人员和需要为视频添加字幕的用户。

## 技术栈

- **核心依赖**: faster-whisper (Whisper 模型的高效实现)
- **音频/视频处理**: moviepy (用于从视频中提取音频)
- **在线视频处理**: yt-dlp (用于解析和下载在线视频内容)
- **编程语言**: Python 3.8+

## 功能特性

- 批量处理本地视频文件，支持多种视频格式 (.mp4, .avi, .mov, .mkv, .wmv, .flv, .webm)
- 自动从视频中提取音频，避免直接处理高显存视频
- 使用 faster-whisper 本地模型进行语音转文本
- 生成带时间戳的 Markdown 格式字幕文件
- 支持不同大小的 Whisper 模型 (tiny, base, small, medium, large)
- 支持 CPU 和 GPU (CUDA) 设备
- **新增**: 在线视频 URL 解析支持，优先下载官方字幕（中文/英文）
- **新增**: 无官方字幕时自动下载音频流并进行本地 AI 转录
- **新增**: 支持主流视频网站（YouTube, Bilibili, Vimeo 等）
